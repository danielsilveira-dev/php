03. Linguagem interpretada

Falando um pouco sobre PHP vimos que ele é uma linguagem de programação interpretada. O que isso quer dizer?

Que não é necessário executar um comando para tornar o código executável a cada alteração

O código PHP da forma como escrevemos é interpretado por um programa (chamado de, pasmem, PHP), que executa suas instruções. Na prática, o PHP até transforma o código que escrevemos em um outro tipo de código (bytecode) legível pelo interpretador antes de começar a executar, mas isso é um assunto um pouco mais avançado. :-D

Configuração

Após baixado o arquivo zipado do php, vamos extrair para uma pasta.

  1. Ir em propriedades do Computador
  2. Configurações avançadas do sistema
  3. Variáveis de ambiente
  4. Em "variáveis de usuário para ..."
  5. Selecionar a variável "Path" e clicar em "Editar"
  6. Clique em "Novo" e cole o endereço da pasta do executavel do php

Posso abrir o CMD do windows ou do php e digitar: php -v

Se estiver funcionando, irá aparecer a versão que foi instalada.

php.ini

No arquivo php.ini dentro desta pasta, contém diversas configurações que podemos configurar como: quais extensões vêm habilitadas, se conectar com o banco de dados específico, manipular algum tipo de dado, se os erros devem exibir na mesma tela.

Existem dois arquivos php.ini para configurar. Um para configurar quando estiver em modo de desenvolvimento e outro quando estiver em modo de produção.

Oque iremos editar, podemos deixar somente o nome "php.ini".

Imprimindo uma mensagem na tela

Para imprimir uma mensagem na tela, podemos fazer da seguinte forma:

echo "Olá mundo"

Shell Interativo

Para abrir um shell interativo, no cmd, podemos digitar:

php -a

Irá abrir uma linha de comando para digitação de linguagem php.
